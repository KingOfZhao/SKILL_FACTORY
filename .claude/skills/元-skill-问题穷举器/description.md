### 精炼描述
无限物理实践/Skill 树穷举器：对任意问题进行树状无限穷举真实可动手实践（物理实验）或最小 Skill 结构拆分，支持两种模式切换，上下文即将满额时自动保存节点，下次无缝继续。

### 详细描述

**核心特性：**
- 双模式支持：物理实践穷举 / 最小 Skill 树实践穷举
- 树状无限穷举：横向（同层变体）+ 纵向（基础→进阶→极端）
- 智能持久化：上下文满额时自动保存 checkpoint
- 无缝继续：下次调用自动从上次节点继续
- 10分钟可验证：每个实践都可以在短时间内验证

**两种模式说明：**

1. **物理实践穷举**
   - 重点：真实世界中可动手、可执行、可观察、可重复的具体行动、实验、工具使用、材料组合、环境搭建、流程操作等
   - 强调可操作性、物理世界反馈（重量、时间、声音、温度、视觉变化等）
   - 例子：如何用家用物品验证重力加速度、如何手工制作简易电磁铁等

2. **最小 Skill 树实践穷举**
   - 重点：针对"构建 Skill 体系"本身的最小化、模块化、可组合的 Skill 定义与拆分树
   - 强调单一职责、边界清晰、10分钟可验证、依赖关系、底层约定兼容等
   - 例子：针对某个目标（如"自动化微信小程序测试"），穷举出最小的 Skill 树结构（根 Skill → 子 Skill → 依赖关系）

**技术实现：**
- 零外部依赖（仅需文件系统 MCP）
- JSON 格式 checkpoint
- 自动上下文检测（30% 剩余 / 140k token 阈值）
- 独立 checkpoint 文件避免模式混淆
