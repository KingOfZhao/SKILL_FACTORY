# 无限物理实践/Skill 树穷举器

对任意问题进行树状无限穷举真实可动手实践或最小 Skill 结构拆分。

## 使用方法

### 启动模式选择
首次使用时，系统会询问你要穷举的模式：
- **1. 物理实践** - 真实世界可动手、可观察的具体行动、实验、流程操作
- **2. 最小 Skill 树** - Skill 体系的最小化定义、拆分树、单一职责模块

### 基本用法
```
无限穷举器：[我的问题]
```

### 继续上轮
```
继续
```

或再次输入同一个问题即可。

## 输出内容

每轮输出包含：
- 5-15 个实践列表（编号 + 描述 + 执行步骤 + 预期观察 + 耗时/成本）
- 树状结构预览（当前节点路径）
- 更新后的 checkpoint 文件
- 10 分钟验证指南

## Checkpoint 机制

两种模式使用独立文件：
- 物理实践：`enumeration_checkpoint_physical.json`
- 最小 Skill 树：`enumeration_checkpoint_skilltree.json`

上下文即将满额时自动保存，下次无缝继续。

## 10 分钟验证

1. 查看 checkpoint 文件（<10秒）
2. 阅读实践列表（3分钟）
3. 任选 1 个实践尝试（5分钟）
4. 对比结果判断正确性

## Limitations

- 只负责穷举，不负责执行或结果解读
- 每次仅输出有限 chunk
- 依赖文件系统 MCP
- 抽象问题可能输出偏向可动手实验
