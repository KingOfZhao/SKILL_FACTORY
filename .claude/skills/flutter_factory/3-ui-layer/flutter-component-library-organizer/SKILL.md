# 组件库组织器（Component Library Organizer）

## Capabilities（单一职责）
- 根据组件列表组织组件库目录
- 生成组件文档索引
- 输出库结构和文档

## 执行流程（5 步骤）
1. 解析组件列表
2. 组织目录结构
3. 生成文档索引
4. 创建 README
5. 输出到 output/library/

## 10 分钟验证指南
运行组织器 → 检查目录结构 → 验证文档完整性

## Limitations
只组织结构，不实现组件
