# 图像转规格转换器（description.md）

## 核心问题
设计师提供 UI 截图或草图，开发人员需要将视觉设计转换为技术规格。手动描述耗时且容易遗漏细节。

## 问题背景
1. UI 设计以图像形式呈现
2. 开发需要知道元素位置、尺寸、样式
3. 手动标注工作量大
4. 图像分析技术已成熟

## 解决方案
使用图像分析技术自动提取规格：
- 识别 UI 元素类型（按钮、输入框、列表等）
- 提取位置和尺寸信息
- 提取样式属性（颜色、字体、圆角等）
- 生成结构化技术规格

## 输入
UI 截图或草图（.png, .jpg）

## 输出
图像技术规格：
- ui_elements：UI 元素列表
- dimensions：图像尺寸
- layout_suggestions：布局建议

## 使用场景
1. UI 设计图转技术规格
2. 设计师与开发者协作
3. 快速原型设计验证
4. 设计规范文档生成

## 预期效果
开发者可以在 10 分钟内：
1. 上传 UI 设计图
2. 获得技术规格 JSON
3. 验证元素识别准确性
4. 用于 Flutter 开发

## 已知局限
- 依赖图像清晰度
- 手绘草图识别精度低
- 不识别复杂交互
- 不生成实际代码
