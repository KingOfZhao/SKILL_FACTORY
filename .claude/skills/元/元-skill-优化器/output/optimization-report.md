## 修复器增强

详见: [fixer-enhancement.md](fixer-enhancement.md)

### 主要增强内容

#### 新增检测维度（4个）
1. **代码质量检查** - 重复代码、复杂度、代码风格
2. **安全性检查** - 敏感信息、命令注入、权限检查
3. **国际化检查** - 硬编码文本、语言特定格式
4. **可维护性检查** - 文档完整性、版本管理、依赖声明

#### 修复逻辑增强（3个）
1. **智能修复推荐** - 基于历史数据推荐修复策略
2. **上下文感知修复** - 考虑 Skill 类型、使用场景、依赖关系
3. **冲突解决** - 自动/半自动解决约定、版本、资源冲突

#### 输出格式优化（2个）
1. **增强 JSON 输出** - 包含置信度、可自动修复标识
2. **可视化报告** - HTML 报告 + 图表 + 交互功能

#### 性能优化（3个）
1. **并行检查** - 同时检查多个文件
2. **结果缓存** - 缓存检查结果
3. **增量检查** - 只检查变更文件

1. **质量保障** - 增加 validator 和 test-gen
