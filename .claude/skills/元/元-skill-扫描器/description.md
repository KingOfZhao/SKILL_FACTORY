# Skill 目录扫描器（description.md）

## 核心问题
Claude Code 和用户需要一个快速、自动化的方式来了解某个文件夹下有哪些 skills，每个 skill 的能力是什么，以及它们之间的目录结构关系。

## 问题背景
1. 当前 skills.json 仅记录了部分技能，且格式不适合直接用于树状结构展示
2. 用户需要快速浏览技能目录结构，了解技能组织方式
3. Claude Code 需要程序化地读取技能信息，以便根据路径进行调用
4. 没有统一的工具可以扫描和生成能力清单

## 解决方案
提供一个自动化扫描工具：
- 扫描指定文件夹下的所有 skill 目录
- 解析每个 skill 的 SKILL.md 获取基础信息
- 生成人类可读的 Markdown 清单
- 生成机器可读的 JSON 清单
- 输出树状结构图

## 输入
- 目标文件夹路径（绝对或相对路径）
- 可选参数：仅输出 JSON

## 输出
1. Markdown 格式能力清单（含树状结构图）
2. JSON 格式清单（含完整 skill 信息）
3. 扫描统计信息

## 使用场景
1. 用户快速了解某目录下有哪些 skills
2. Claude Code 读取技能信息并按需调用
3. 技能目录结构分析和可视化
4. 技能清单的快速更新和维护

## 预期效果
用户和 Claude Code 可以在 10 分钟内：
1. 运行扫描器
2. 查看 Markdown 清单了解所有技能
3. 使用 JSON 清单进行程序化调用
4. 验证扫描结果的准确性

## 已知局限
- 不执行 skill 内容验证
- 依赖 SKILL.md 规范性
- 不反映技能间的依赖关系
- 扫描受文件系统权限限制
