# 元-Skill 检查器

## 描述

元-Skill 检查器是一个验证工具，用于检查所有非元-前缀的 skill 是否符合 10 分钟可验证原则。

## 核心功能

1. **自动扫描** - 扫描 skills 目录，自动识别非元-前缀 skill
2. **五维验证** - 从文件完整性、内容结构、验证机制、失败处理、输出可验证性五个维度验证
3. **JSON 输出** - 生成结构化的 JSON 报告
4. **清晰建议** - 为每个不合规 skill 提供具体的改进建议

## 使用场景

- 定期检查技能库的合规性
- 新增 skill 后的合规验证
- 技能库质量审计

## 设计原则

- 只检查，不修改
- 元-skill 自动豁免
- 静态分析为主，确保快速执行
